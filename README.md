# CrunchyRoll_History_extract
CrunchyRoll_History_extract

>Les appels d'API sont trop compliqués.
>les addons dépendent du navigateur
>les scripts powershell dépendent d'un windows.
>
>Extraction manuelle de l'historique en JSON
>Analyse et mise en forme avec page HTML+Javascript des JSON.
>
>1 - Avec le navigateur (FireFox pour ma part)
>2 - F12
>3 - Onglet réseau
>4 - Filtrer sur "watch-history"
>5 - Aller sur https://www.crunchyroll.com/fr/history
>6 - Sur la page https://www.crunchyroll.com/fr/history descendre tout en bas (faire Page Down sur le clavier pour plus d'efficacité)
>7 - Dans le F12, Chaque chargement de page, ajoute une ligne, sur chaque ligne présente : Clic droit / Enregistrer la réponse sous (mettre dans le même répertoire)
>
>Utiliser la page HTML (et son javascript inclus) pour glisser/déposer les fichiers les fichier récupérés et faire la mise en page.
